# MineSweeper

MineSweeper game project for [PROGRAMMING PART 2](https://www.unive.it/data/insegnamento/230202) - University of Venice.

Execution: 
  - Clone the project, open a terminal and navigate to the directory
  - Type:
    ```bash
    make CampoMinato
    ./CampoMinato
    ```

Made in 2016/2017.
